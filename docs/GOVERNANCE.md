# CeCell Governance  
### Roles • Responsibilities • Decision‑Making • Community Standards

CeCell follows a lightweight, transparent governance model aligned with the Ascendia mission.

---

# 🧭 Roles

### **Project Lead**
- Oversees roadmap  
- Approves major changes  
- Maintains vision alignment  

### **Core Maintainers**
- Review PRs  
- Manage releases  
- Ensure code quality  

### **Contributors**
- Submit PRs  
- Improve docs  
- Report issues  
- Suggest features  

---

# 🗳️ Decision‑Making

### Minor Changes
Handled by maintainers.

### Major Changes
Require:
- Proposal  
- Discussion  
- Approval from Project Lead  

### Breaking Changes
Require:
- Formal review  
- Documentation updates  
- Version bump  

---

# 📜 Standards

- Accessibility is non‑negotiable  
- Code must be maintainable  
- Documentation must be clear  
- Features must align with CeCell’s mission  

---

# ❤️ Community

We welcome contributors who share our values:
- empathy  
- clarity  
- accessibility  
- empowerment
