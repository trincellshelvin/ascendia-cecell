## [1.0.0] — Initial Release
### Added
- Core CeCell extension framework  
- Command Router subsystem  
- Patch Engine (file + workspace patching)  
- Merge Engine (simulation + prediction)  
- Workspace Analyzer (indicators + structure + health)  
- Memory Subsystem (write, search, retrieve)  
- Autonomous Panel  
- Memory Panel  
- Workspace Intelligence Panel  
- Voice Command activation  
- Full documentation suite (README, ARCHITECTURE, COMMANDS, DEVELOPMENT, CONTRIBUTING, API, ROADMAP)

### Notes
This release establishes CeCell as an accessibility‑first, voice‑driven, autonomous developer assistant — the foundation of the Ascendia developer ecosystem.
