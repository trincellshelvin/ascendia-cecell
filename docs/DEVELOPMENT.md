# CeCell Development Guide  
### Build • Test • Package • Debug • Contribute

This guide explains how to develop, build, test, and package the CeCell VS Code extension.  
It is written for contributors, maintainers, and developers integrating CeCell into the Ascendia ecosystem.

---

# 🧱 Prerequisites

Before developing CeCell, ensure you have:

### **Required Tools**
- Node.js (LTS recommended)
- npm
- Visual Studio Code
- TypeScript
- VSCE (Visual Studio Code Extension Manager)

Install VSCE globally:

```bash
npm install -g @vscode/vsce
