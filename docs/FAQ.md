# CeCell FAQ  
### Common Questions • Quick Answers • Developer Guidance

---

# ❓ What is CeCell?
An accessibility‑first, voice‑driven, autonomous developer assistant for VS Code.

---

# ❓ Why is it named CeCell?
It honors Granny Cecell — a legacy of clarity, strength, and guidance.

---

# ❓ Does CeCell require the cloud?
No.  
CeCell works fully offline.  
Cloud features are optional.

---

# ❓ Can I add my own commands?
Yes — see `CONTRIBUTING.md` and `API.md`.

---

# ❓ Does CeCell modify my files?
Only when you:
- apply a patch  
- run a fix command  
- confirm an action  

CeCell never changes files silently.

---

# ❓ How do I debug CeCell?
Use:
- `CeCell: Debug Mode`  
- `CeCell: Show Logs`  
- VS Code Developer Tools  

---

# ❓ Can CeCell work across multiple repos?
Yes — multi‑repo intelligence is part of the roadmap.

---

# ❓ How do I get help?
See `SUPPORT.md`.
