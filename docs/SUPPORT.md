# CeCell Support Guide  
### Documentation • Questions • Future Availability

CeCell is currently in **documentation‑only mode**.  
The assistant itself, installation packages, and any subscription options are not yet available for download or purchase.

---

# 🧾 Current Status

- CeCell is in **design and documentation** phase.  
- The implementation, packaging, and distribution are being prepared privately.  
- Public access will be announced once the broader Ascendia platform is ready.

---

# ❓ What can I do right now?

You can:

- Read the documentation in the `docs/` folder  
- Learn about CeCell’s architecture, commands, and roadmap  
- Plan how CeCell might fit into your future workflows  

You **cannot** (yet):

- Download CeCell  
- Install a VSIX or extension package  
- Purchase a subscription  
- Access any cloud features
